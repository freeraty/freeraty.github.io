Como funciona o freeraty.com?

    O freeraty.com é um site para procura e divulgação de trabalhos freelancers, inicialmente voltados para a área de programação e design. O website tem a ambição de conectar clientes e profissionais sem que para isso tenha que ser cobrados taxas ou comissões para o freelancer. Todo o processo funciona com cadastro de e-mail, tanto do profissional quanto dos clientes. Assim que um projeto é publicado no sistema, os profissionais podem enviar suas propostas para o sistema para o cliente avaliar.Após todos os detalhes do projeto estarem combinados, é dado início ao projeto. Todo o processo de entregas de etapas e pagamentos são registrados no sistema por clientes e profissinais para uso posterior de suas reputações.  Clique abaixo em um dos botões que representa o seu caso!
